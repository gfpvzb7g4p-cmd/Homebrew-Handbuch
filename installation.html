Homebrew – Installation & Grundbefehle

1. Installation

Installation
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Offizielles Installationsskript. Apple‑Silicon legt /opt/homebrew an.

2. Shell‑Einbindung

Installation
echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zprofile
Installation
eval "$(/opt/homebrew/bin/brew shellenv)"

Erste Zeile dauerhaft, zweite sofort aktiv.
